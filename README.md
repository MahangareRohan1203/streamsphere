# StreamSphere - Scalable Video Streaming Platform

## Tech Stack
- Java, Spring Boot, Spring Cloud
- OAuth2, JWT
- Kafka
- Redis
- PostgreSQL, MongoDB
- React (Frontend)
- Docker, Kubernetes

## Architecture
Microservices-based architecture with API Gateway, Auth Service, Video Service, and Recommendation Engine.

## Features
- OAuth Authentication
- Video Streaming (CDN simulated)
- Resume Playback
- Recommendation System
- Event-driven architecture using Kafka
